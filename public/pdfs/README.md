# PDF Fayllar

Bu papkaga ma'ruza matnlarini PDF formatida joylashtiring.

## Fayl nomlash qoidalari:

PDF fayllarni quyidagi formatda nomlang:

```
1-mavzu.pdf   (1-chi mavzu uchun)
2-mavzu.pdf   (2-chi mavzu uchun)
3-mavzu.pdf   (3-chi mavzu uchun)
...
20-mavzu.pdf  (20-chi mavzu uchun)
```

## Misol:

```
public/
  pdfs/
    1-mavzu.pdf
    2-mavzu.pdf
    3-mavzu.pdf
    ...
    20-mavzu.pdf
```

PDF fayllarni shu papkaga joylashtirgandan so'ng, "Ma'ruza matni" tugmasini bosganingizda PDF viewer ochiladi.
