# Taqdimotlar (Presentations)

Bu papkaga taqdimotlarni **PDF** formatida joylashtiring.

## Fayl nomlash qoidalari:

PDF fayllarni quyidagi formatda nomlang:

```
1-mavzu-taqdimot.pdf   (1-chi mavzu uchun)
2-mavzu-taqdimot.pdf   (2-chi mavzu uchun)
3-mavzu-taqdimot.pdf   (3-chi mavzu uchun)
...
20-mavzu-taqdimot.pdf  (20-chi mavzu uchun)
```

## PowerPoint → PDF konvertatsiya:

1. PowerPoint faylini oching
2. **File → Save As → PDF** yoki **File → Export → Create PDF**
3. Sifatni **High Quality** qilib tanlang
4. PDF faylni shu papkaga saqlang

## Afzalliklar:

- ✅ 100% offline ishlaydi
- ✅ Electron .exe da mukammal ishlaydi
- ✅ Barcha platformalarda qo'llab-quvvatlanadi
- ✅ Tez yuklanadi va ishonchli
- ✅ Mavjud PDF viewer ishlatiladi
